**Beverage Machine Simulation using Python (OOP)**
**Project Overview**

This repository contains a collection of Python simulations of beverage and vending machines, designed to demonstrate how Object-Oriented Programming (OOP) can be used to model real-world systems.

The project explores how machines like juice makers, coffee machines, and vending systems manage menus, resources, and payment processing through modular Python classes.

Each implementation showcases different ways to structure logic and apply OOP principles such as encapsulation, modularity, 
and class interaction.

**Project Objectives**

The goal of this project is to:

1. Practice Object-Oriented Programming in Python
2. Simulate real-world machine systems
3. Design modular and reusable classes
4. Understand how components interact within a system

Build structured programs that are easy to maintain and extend.

**Project Implementations**
This repository contains multiple implementations that simulate different types of machines.

**Juice Machine Simulation**
A simple program that simulates a juice-making machine where users can select drinks and the machine checks ingredient availability before preparing the beverage.

Key Features:

1. Displays available juice options
2. Checks ingredient resources
3. Processes drink selection
4. Simulates drink preparation

This implementation focuses on basic system simulation and program flow control.

**Coffee Machine (OOP Version)**
This version models a coffee machine using Object-Oriented Programming.

The machine is divided into multiple components represented as classes:

1. Menu – Stores available drink options
2. MenuItem – Represents each drink and its ingredients
3. CoffeeMaker – Handles ingredient resources
4. MoneyMachine – Manages payments and tracks profit

This design demonstrates modular architecture and class interaction, where each component handles a specific responsibility.

**Vending Machine Algorithm**
This implementation focuses on the logic behind vending machines, including:

1. Item selection
2. Resource validation
3. Payment handling
4. Transaction processing

The algorithm simulates how vending machines manage inventory, user choices, and transactions in a structured workflow.

This project demonstrates several fundamental programming concepts:

2. Classes and Objects
3. Encapsulation

**Example Program Flow**
1. The system displays available beverage options
2. The user selects a drink
3. The machine checks if ingredients are available
4. The machine processes payment
5. If successful, the drink is prepared and served
